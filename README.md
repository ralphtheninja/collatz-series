# collatz-series

Calculates the Collatz series given a positive integer.

### api

```js
const collatz = require('collatz-series')
console.log(collatz(44))
// [ 44, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1 ]
```
